WordPress Access Areas
======================

About
-----
Allows You to set reading, editing and commenting permissions. 


Compatibility
-------------
- WP 3.5.2 through 3.6
- Works with Multisite
- Translation ready

Installation
------------
Move the plugin dir in your `wp-content/` directory and activate it in the GUI.
